# 3. 경사하강법 (Gradient Descent)
딥러닝의 심장이라고 불리는 경사하강법은 오차를 줄이기 위해 $\mathbf{w}$와 $b$을 조금씩 수정해 나가는 최적화 루프이다. <br>

손실 함수 $L$을 가중치 $\mathbf{w}$로 미분한 것을 **기울기(Gradient)**라고 한다. 즉, 가중치를 1만큼 키웠을 때 오차가 얼마나 변하는지 보여주는 척도이다. <br>

안대를 쓴 상태로 산 골짜기에 있다고 가정해보자. 지면으로 가기 위해 산의 경사를 발로 느껴보면서, 발바닥이 땅에 닿는 면적 중 가장 낮은 쪽을 찾는 것이다. 

우리는 손실을 줄여야 하니까 기울기가 가리키는 방향의 반대방향으로 가중치를 이동시켜야 한다.
- 기울기(Gradient)가 플러스($+$)라면, 가중치를 키웠을 때 오차가 커진다는 뜻이므로 가중치를 줄여야 한다.
- 기울기(Gradient)가 마이너스($-$)라면, 가중치를 키웠을 때 오차가 작아진다는 뜻이므로 가중치를 키워야 한다. <br>

모델은 매 단계마다 아래 식을 통해 스스로 업데이트한다.
$$(\mathbf{w},b) \leftarrow (\mathbf{w},b) - \frac{\eta}{|\mathcal{B}|} \sum_{i \in \mathcal{B}_t} \partial_{(\mathbf{w},b)} l^{(i)}(\mathbf{w},b)$$
- $\mathcal{B}$ (Minibatch): 현재 계산에 사용하는 데이터 묶음.
- $|\mathcal{B}|$: 배치 크기(데이터 개수)로 나눠서 평균을 낸다.
- $\eta$ (Learning Rate, 학습률): 한 번에 얼마나 큰 발걸음으로 이동할지를 결정하는 숫자이다.
<br>

여기서 *배치(Batch)**라는 단어가 나오는데, 배치는 딥러닝에서는 데이터를 처리하는 기본 단위를 의미한다.<br>

딥러닝 모델은 보통 병렬 처리를 하는 GPU에서 돌아가는데, GPU 메모리(VRAM)에는 당연히 한계가 존재한다. 데이터 개수가 적게는 수백만, 많게는 수억 개라면, 한 번 업데이트하기 위해 수많은 데이터를 한 번에 다 계산하는 건 아무리 좋은 GPU를 쓰더라도 매우 느릴 것이다. 반대로 데이터를 1개씩 업데이트 한다면 노는 시간(Idle)이 너무 많아지고 데이터 하나하나에 너무 민감해서 비효율적이다.<br>

그렇기 때문에 데이터 묶음(Minibatch)를 써서 GPU의 병렬 처리 능력을 효율적으로 써먹는 것이다. 보통 데이터 1개를 처리하는 시간과 32개를 처리하는 시간이 거의 비슷하기 때문에 보통 32개의 묶음으로 배치 사이즈를 설정해서 쓴다. 이건 상황에 따라 유동적으로 설정하지만 보통 $2^n$개 단위로 잡는다.<br>
(GPT, 제미나이 같은 거대한 LLM 모델은 배치 사이즈를 엄청 크게 키우려고 여러 대의 GPU에 나눠서 계산하는 분산학습을 하기도 하는데, 이런 경우를 제외하면 보통은 소규모 배치로 처리한다.)
<br>

다시 안대를 쓴 상태로 산 골짜기에 있다고 가정해보자. 지면으로 가기 위해 산의 경사를 발로 느껴보면서, 발바닥이 땅에 닿는 면적 중 가장 낮은 쪽을 찾는 것이라고 했다. 이때 배치라는 개념은 혼자 내려가는 것이 아니라 근처에 있는 사람 32명(Batch Size)에게 어디가 낮은지 물어보는 것이다.
<br><br>

### 하이퍼파라미터 (Hyperparameters)
다시 돌아와서 가중치($\mathbf{w}, b$)는 모델이 알아서 학습하지만, 배치 크기($\mathcal{B}$)와 학습률($\eta$)은 직접 설정해줘야 한다. 이걸 하이퍼파라미터라고 부른다.
- 학습률($\eta$)이 너무 크면? 최저점을 지나쳐서 튕겨 나간다. (Over shooting)
- 학습률($\eta$)이 너무 작으면? 기어가는 수준이라 학습이 영원히 안 끝난다.
<br><br>

학습률은 산에서 내려갈 때 보폭을 의미한다. 경사가 아래인 걸 파악했어도 얼마나 크게 움직일지가 중요하다. 보폭이 너무 크면 지면을 밟지도 못하고 반대편 산 골짜기로 넘어가버릴 것이다. 반대로 보폭이 너무 작으면 평생 지면 근처에도 못가게 될 것이다. 그렇기에 이 보폭(학습률)을 얼마나 할지가 중요하다.
