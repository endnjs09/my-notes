# 2. 손실함수(Loss Function)

손실함수는 실제 정답($y$)과 모델의 예측값($\hat{y}$) 사이의 거리(Distance)를 측정하는 것으로, 쉽게 말해 '내 예측이 얼마나 틀렸는지' 점수를 매기는 법이다. 손실값이 0에 가까울수록 "공부 잘했다"는 뜻이고, 값이 클수록 "엉터리다"라는 뜻이다.
<br>

### 1) 제곱 오차 (Squared Error)
회귀 문제에서 가장 흔하게 쓰이는 방식으로, 모델이 데이터 하나(i번째)에 대해 얼마나 틀렸는지 계산하는 방법이다.
$$l^{(i)}(\mathbf{w}, b) = \frac{1}{2}(\hat{y}^{(i)} - y^{(i)})^2$$
제곱을 하는 이유는 단순히 더하기 빼기만 하면 오차가 음수나 양수가 섞여서 합쳤을 때 0이 될 수 있다. 제곱을 하면 모든 오차가 양수가 되어 정확한 거리를 측정할 수 있게 된다.
<br><br>

### 2) 평균 제곱 오차(MSE)
데이터 하나만 잘 맞추는 게 아니라 $n$개의 데이터 전체를 잘 맞춰야 한다. 그래서 모든 데이터 샘플의 손실을 더한 뒤 평균을 낸다.
$$L(\mathbf{w}, b) = \frac{1}{n} \sum_{i=1}^n l^{(i)}(\mathbf{w}, b) = \frac{1}{n} \sum_{i=1}^n \frac{1}{2}(\mathbf{w}^\top \mathbf{x}^{(i)} + b - y^{(i)})^2$$
제곱을 해주는 또 다른 이유는 오차가 클수록 제곱 값이 기하급수적으로 커지는데, 모델에게 작은 실수 여러 번보다 큰 실수 한 번이 더 치명적이라고 가르치기 위함이다. 하지만 이상치에 너무 민감하게 반응해서 전체 모델이 망가질 수 있다는 단점도 있다.<br>

추가적으로 위 식에서 $n$으로 나눠주는 이유는 데이터가 많아지면 오차 총합이 커져 모델이 당황하게 하는 것을 막기 위함이다. $n$으로 나누지 않을 경우 모델의 실력은 그대로인데 데이터 샘플 개수만 늘어나도 오차가 훅 치솟는다.
<br><br>

### 3) 최적화
결국 우리가 하고 싶은 건, 이 손실 함수 $L(\mathbf{w}, b)$를 최소로 만드는 가중치($\mathbf{w}^*$)와 편향($b^*$)을 찾는 것이다.
$$\mathbf{w}^*, b^* = \underset{\mathbf{w}, b}{\operatorname{argmin}} L(\mathbf{w}, b)$$
$\operatorname{argmin}$ 즉, 함수값을 최소로 만다는 인자(parameter)를 찾으라는 뜻으로, 무수히 많은 학습을 반복하면서 오차(Loss)를 줄이기 위해  $\mathbf{w}$와 $b$를 수정해 나가는 것이다.
