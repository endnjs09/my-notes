## 왜 미분을 공부해야 하는지?
가중치를 아주 조금 바꿨을 때 오차가 얼마나 민감하게 바뀌는 가를 알아내기 위해서이다. 미분 값이 양수인지 음수인지에 따라 가중치를 설정하는 경사하강법의 핵심이기 때문에 딥러닝에서 미분은 필수적이다.

# 1. 미분 (Derivatives)
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$
- 입력 $x$를 아주 미세하게($h$) 변화시켰을 때, 결과$f(x)$가 얼마나 변하는지의 비율이다.
    ```python
    def f(x):
        return 3 * x ** 2 - 4 * x
    
    for h in 10.0**np.arange(-1, -6, -1):   # -1에서 -6까지 1씩 감
        print(f'h={h:.5f}, numerical limit={(f(1+h)-f(1))/h:.5f}')
    ```
    ```
    h=0.10000, numerical limit=2.30000
    h=0.01000, numerical limit=2.03000
    h=0.00100, numerical limit=2.00300
    h=0.00010, numerical limit=2.00030
    h=0.00001, numerical limit=2.00003
    ```

# 2. 편미분(Partial Derivatives)
딥러닝 모델은 입력 변수가 적게는 수만개에서 많게는 수백, 수천만 개이다 ($y = f(x_1, x_2, \dots, x_n)$). 이때 특정 변수 하나에 대해서만 미분하는 것을 편미분이라고 한다.

$$\frac{\partial y}{\partial x_i} = \lim_{h \rightarrow 0} \frac{f(x_1, \ldots, x_{i-1}, x_i+h, x_{i+1}, \ldots, x_n) - f(x_1, \ldots, x_i, \ldots, x_n)}{h}$$
-  관심 있는 변수 $x_i$를 제외한 나머지 모든 변수는 상수 취급하고 미분한다.
<br><br><br>

# 3. 기울기 (Gradients)
편미분 값들을 일렬로 세운 벡터로 모아둔 것이다. Gradients 라는 단어는 앞으로도 자주 보게 될 것이다.  
$$\nabla f(\mathbf{x}) = \left[ \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \dots, \frac{\partial f}{\partial x_n} \right]^\top$$
- 함수 값이 가장 가파르게 증가하는 방향을 가리키는데, 반대로 Gradient의 반대 방향 ($-\nabla f$)으로 가면 함수 값이 가장 빠르게 감소한다. 이게 바로 경사하강법의 핵심원리이다.
<br><br><br>

# 4. 연쇄 법칙 (Chain Rule)
합성함수를 미분하는 방법으로 딥러닝의 **역전파(Backpropagation)**의 핵심이다.
$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$$
- $y = f(u)$ 이고 $u = g(x)$ 일 때, $x$가 $y$에 미치는 영향력을 구하고 싶다면 중간 단계($u$)의 미분값을 곱해주는 방식이다.
- 쉽게 말해 $x$가 변할 때 $u$가 얼마나 변하는지 측정하고, 그 변화가 $y$에 얼마나 영향을 주는지 재서 둘을 곱하면 된다.
