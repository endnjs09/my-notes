# 0. 개요
2-3, 2-4의 미분은 '어떻게 학습시킬 것인가' 방향에 다룬다면 확률(통계)는 '데이터의 불확실성을 어떻게 숫자로 다룰 것인가'에 대해 다룬다.
예를 들어,
- 이 사진이 개일 확률은 95%이고 고양이일 확률은 5%이다.
- 동전 앞면이 나올 확률은 50%이다.
<br>
### 동전 던지기
우리는 동전 앞면 혹은 뒷면이 나올 확률이 0.5라고 당연히 알고 있다. 하지만 수학적으로는 어떻게 증명할까?
동전을 수없이 많이 던져서 앞면이 몇 번 나오는지 직접 세보면 될 것이다. 동전을 $n$번 던졌을 때 앞면($H$)이 $n_H$번 떴다면 확률은 아래와 같다.
$$\hat{P}(X=H) = \frac{n_H}{n}$$
동전을 10번 던졌을 때 앞면이 6번 나올 수 있지만 1000번, 100만번... 무수히 던지게 되면($n$) 결국 기대값($\hat{P}$)은 0.5($P$)에 가까워진다는 게 **대수의 법칙**이다.
Pytorch에서는 Multinomial(다항분포) 패키지를 사용하여 동전 던지기 시뮬레이션을 수행한다
```python
import torch
from torch.distributions import multinomial

fair_probs = torch.ones([2]) / 2  # [0.5, 0.5]
multinomial.Multinomial(10, fair_probs).sample()    # 10번 던졌을 때 
multinomial.Multinomial(10000, fair_probs).sample()    # 10000번 던졌을 때 
```
```python
tensor([4., 6.])
tensor([0.4966, 0.5034])
```
<br>
시행 횟수를 늘리면 실제 확률가 가까워지는지 확인하기 위해 그래프로 확인해보자.

```python
import torch
from torch.distributions import multinomial
import matplotlib.pyplot as plt

# 데이터
fair_probs = torch.tensor([0.5, 0.5])
counts = multinomial.Multinomial(1, fair_probs).sample((10000,))
cum_counts = counts.cumsum(dim=0)
estimates = cum_counts / cum_counts.sum(dim=1, keepdims=True)
estimates = estimates.numpy()

# 그래프 그리기
plt.figure(figsize=(10, 6))
plt.plot(estimates[:, 0], label='P(coin=heads)', color='#1f77b4') # 파란색
plt.plot(estimates[:, 1], label='P(coin=tails)', color='#ff7f0e') # 주황색
plt.axhline(y=0.5, color='black', linestyle='dashed', alpha=0.5)
plt.xlabel('Samples (Number of tosses)')
plt.ylabel('Estimated probability')
plt.title('Law of Large Numbers: Convergence of Coin Tosses')
plt.legend()
plt.grid(True, alpha=0.3)

plt.show()
```

그래프가 시행횟수가 적을 땐 요동치지만 시행횟수가 늘어날 수록 0.5에 수렴하게 되는 것을 볼 수 있다. 딥러닝 모델이 데이터를 많이 필요로 하는 이유도 설명이 되는데, 데이터가 적으면 위 그래프의 왼쪽 부분처럼 우연에 휘둘리게 될 수 있다.
